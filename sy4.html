实验四：网站页面布局

一、实验目的

  1. 掌握流行的整体布局方式CSS+Div
  2. 掌握局部的表格布局方式
  3. 掌握框架布局、页内框架的使用

二、实验内容及步骤

1. 主页布局效果

  • 主页整体上由“上-中-下”三个等高的div组成，它们在浏览器窗口中均水平居中
  • 主页头部由上下两个div组成，上面div又由三个并排的div组成
  • 中间div是页面的主要部分，由左右并排的两个div组成，右div被定义为一个页内框架，供显示初始页面和链接的目标页面（在超链接里使用target属性、按名引用页内框架）。

2. 会员管理项目主页头部的CSS+Div布局

HTML代码：

<div class="top">
    <div class="row1"> <!-- 第一行 -->
        <div class="row11"><span id="dtps">date and time</span></div>
        <div class="row12">会员管理信息系统</div>
        <div class="row13">尚未登录！</div>
    </div>
    <div class="row2">  <!-- 第二行 -->
        <ul>
            <li><a href="index.html">站点主页</a></li>
            <li><a href="mLogin.html">会员登录</a></li>
            <li><a href="mRegister.html">会员注册</a></li>
            <li><a href="index.html" onclick="return confirm('确实要退出会员登录吗？')">会员登出</a></li>
        </ul>
    </div>
</div>

CSS样式代码：

* {
   box-sizing: border-box; /*消除由于使用padding属性而引起的默认膨胀效果*/
   overflow: hidden; /*不扩大隐藏容器*/
   padding:0;margin:0;
}
a{
   text-decoration: none;
}

.top {
   width: 800px;
   height: 80px;
   margin: 30px auto 0px auto;/*水平居中*/
   /*border:1px red solid;
   background:url(images/bg2.jpg) repeat-x;*/
   background-color: #CC6;
}

.row1 {
   width: 800px;
   height: 50px;
   background-color:rgb(62, 128, 0);
}

.row11 {
   width: 280px;
   height: 50px;
   line-height: 50px;/*行高*/
   text-align: center;
   float: left; /*浮动属性，实现并排*/
}

.row12 {
   width: 370px;
   height: 50px;
   line-height: 50px;/*行高*/
   font-size: 30px;
   font-family: "方正舒体";
   float: left; /*浮动属性，实现并排*/
}

.row13 {
   width: 150px;
   height: 50px;
   text-align: center;/*左右居中*/
   color: #FF0000;
   line-height: 50px;/*行高*/
   float: left; /*浮动属性，实现并排*/
   /*border:1px red solid;*/
}
.row2 { /*菜单*/
   width: 800px;
   height: 30px;
   /*border:1px red solid;*/
   line-height: 30px;
   padding-left: 130px;
}
.row2 ul{
   display: inline;/*非常重要，列表默认是块级元素（block），即会产生换行，此处设定为行内样式inline*/
   list-style: none;  /*取消项目符号*/
}
.row2 ul li {
   width: 150px;
   float:left; /*并排列表项*/
}
.row2 ul li a {
   font-size: 20px;
   color: #666666;
   text-decoration: none;
}
.row2 ul li a:hover {  /*使用空格建立的上下文样式，实现精准控制*/
   color: #FF0000;
   /*text-decoration: underline;*/
}
  • *样式是公共样式
  • 标签样式相当于重新定义HTML元素的默认外观
  • 对div应用背景图像很常用

3. 会员管理项目主页主体的CSS+Div布局

HTML代码：

<!-- 主页主体内容 -->
<div class="main">
    <div class="left">
        <br>
        <center style="line-height:50px;font-size:30px">技术文档</center>
        <ul>
            <li><a href="document/1.html" target="kj">1.使用VS Code和WAMP</a></li>
            <li>2.使用HTML标签组织页面内容</li>
            <li>3.使用CSS样式修饰页面元素</li>
            <li>4.网页布局</li>
            <li>5.使用客户端脚本实现页面的交互效果或动态效果</li>
            <li>6.HTML 5</li>
            <li>7.使用优秀的Web前端框架（如LayUI、BootStrap等）实现快速开发</li>
            <li><a href="http://www.baidu.com" target="kj">8.百度一下</a></li>
        </ul>
    </div>
    <!-- 显示详细新闻内容 -->
    <div class="right">
        <iframe name="kj" width="550px" height="550px" src="index0.html" frameborder="no"></iframe>
    </div>
</div>

CSS样式代码：

.main {
    width: 800px;
    height: 550px;
    margin: 0 auto;/*水平居中*/
    background: url(../images/bg.jpg) repeat-x;
    /*border:1px red solid;*/
}
.main ul li{
    line-height: 40px;
}
.left {
    width: 250px;
    height: 550px;
    float: left;
    /*浮动实现并排*/
    background: url(images/bg.jpg);
    /*text-align: center;*/
}
.left ul {
    list-style: none;  /*取消列表的符号*/
}

.left li {
    line-height: 20px;
    padding-left: 15px;
    font-size: 15px;
}

.right {
    width: 550px;
    height: 550px;
    float: left;  /*浮动实现并排*/
}
  • main div由左right与右right两个div并排
  • 左div里的列表采用默认的上下排列
  • 右div作为一个页内框架，作为超链接的目标显示区域

4. 会员管理项目注册页面主体的表格布局

HTML代码：

        <table width="450" border="0">
            <caption style="line-height:45px;font-size:35px;color:red">会员注册</caption>
            <tr>
                <td align="right">会员名称：</td>
                <td><input type="text" id="username" value=""></td></tr>
            <tr>
                <td align="right">会员密码：</td>
                <td><input type="password" id="password" value=""></td></tr>
            <tr>
                <td align="right">性&nbsp;&nbsp;&nbsp;&nbsp;别：</td>
                <td><input type="radio" name="sex" checked value="1">男&nbsp;&nbsp;
                    <input type="radio" name="sex" value="0">女</td></tr>

            <tr>
                <td align="right">兴趣爱好：</td>
                <td><input type="checkbox" value="ah01" name="ah">文艺&nbsp;
                    <input type="checkbox" value="ah02" name="ah">体育&nbsp;
                    <input type="checkbox" value="ah03" name="ah">游戏&nbsp;</td></tr>
            <tr>
                <td align="right">所在城市：</td>
                <td><select name="city" id="city"><option value="0">--请选择--</option></select></td></tr>                 

            <tr>    
                <td align="right">上传照片：</td>
                <td><input type="file" id="phote"></td></tr>    

            <tr>    
                <td colspan="2"><textarea name="jl" id="jl" cols="60" rows="5">这是一个使用textarea标签制作的多行文本框，用于写个人简历之类的东东。</textarea></td></tr>

            <tr height="50">
                <td align="right"><input type="button" id="ensure" value="确定" /></td>
                <td align="center"><input type="button" id="cancel" value="取消" /></td></tr></table>
  • 设置表格标签table的边线宽度为0（即border="0"），实现不显示线条
  • 使用单元格标签td的colspan属性水平合并单元格
  • 表格宽度既可以使用绝对单位，也可以使用相对单位

三、实验小结

  • 利用OOP思想，应用CSS样式可以理解是对对象属性的设置
  • 页内框架应用于超链接
  • 框架布局是指将页面划分为几个独立的框架，详细用法见教材
