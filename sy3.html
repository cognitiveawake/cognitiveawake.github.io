实验三：使用CSS样式设置页面元素外观

一、实验目的

  1. 掌握CSS样式的定义及多种使用方式（内联样式、内部样式和外部样式）
  2. 掌握CSS样式的盒子模型
  3. 掌握类（点）样式与id（#）样式的使用区别
  4. 修改块级元素默认的样式实现并行
  5. 样式混用时优先级（如果有重复的CSS样式）
  6. 使用上下文样式实现精确的元素样式控制
  7. 掌握伪类样式（如a:hover）的使用

二、实验内容及步骤

1. 制作水平导航菜单

导航菜单设计要点：

  • 菜单项放置在一个项目列表里
  • 对列表项应用属性float:left
  • 上下文样式（实际上是一种作用域）实现对元素外观（含鼠标位于元素上时）的精确控制
  • 建立每个列表项对应的超链接

导航菜单的HTML代码：

    <div class="row2">  <!-- 第二行 -->
        <ul>
            <li><a href="index.html">站点主页</a></li>
            <li><a href="mLogin.html">会员登录</a></li>
            <li><a href="mRegister.html">会员注册</a></li>
            <li><a href="index.html" onclick="return confirm('确实要退出会员登录吗？')">会员登出</a></li>
        </ul>
    </div>

导航菜单用的CSS样式：

.row2 {
    width: 800px;
    height: 30px;
    /*border:1px red solid;*/
    line-height: 30px;
    padding-left: 130px;  /*本div内的列表应用了float:left属性，设置文本对齐属性text-align: center对列表无效*/
}
.row2 ul{  /*使用空格建立的上下文样式，实现精准控制*/
  display: inline;/*非常重要，列表默认是块级元素（block），即会产生换行，此处设定为行内样式inline*/
  list-style: none;  /*取消项目符号*/
}
.row2 ul li {
  width: 150px;
  float:left; /*并排列表项*/
}
.row2 ul li a {
  font-size: 20px;
  color: #666666;
  text-decoration: none;
}
.row2 ul li a:hover {  /*使用空格建立的上下文样式，实现精准控制*/
  color: #FF0000;
  /*text-decoration: underline;*/
}

2. 掌握使用遮罩效果的页面中的CSS样式

设计说明： 打开页面后，先单击左边的链接访问新浪网，然后单击页面上方的“登录”超链接，此时将弹出事先隐藏在页面里的“登录”框，而不是新开窗口打开另一个页面。“登录”框出现在半透明的遮罩上，该遮罩使得页面上的超链接元素暂时不可用。在登录操作结束（即单击“确定”或“取消”按钮）后，遮罩和登录框就消失（注册操作也类似）。

                <tr><td width="25%">密&nbsp;&nbsp;码：</td><td width="75%">
                        <input id="pass" maxlength="20" type="password" name="pass"></td></tr>
                <tr><td colspan="2" align="center"><img src="images/ensure.jpg">&nbsp;&nbsp;
                                    <img src="images/cancel.jpg" class="cancel"></td></tr></table></div>

                <caption>用户注册</caption>

三、实验小结

  • 利用OOP思想，应用CSS样式可以理解是对对象属性的设置
  • CSS是独立于HTML的技术，以更加灵活和全面的方式来控制页面元素的外观
  • 上下文样式用于页面元素外观的精准控制
  • 类（点）样式与id样式的区别
  • 使用外部样式的优点（统一外观、多页面共享）
  • CSS样式分类：大小、对齐、边框、显示（或隐藏）、定位（相对与绝对）
