2. 模块化框架Layui的使用

2.1 Layui框架的两种使用方式、使用layer模块做快显信息

快速信息与普通的警告信息不同，它在短暂停留后自动消失，浏览效果如下：

源代码如下：

<!doctype html>
<html>

<head>
    <meta charset="utf-8">
    <title>使用Layui之layer模块做快显信息</title>
    <link rel="stylesheet" href="layui/css/layui.css">
</head>

<body>
<!--script src="layui/layui.all.js"></script-->  <!--非模块化方式-->
<script src="layui/layui.js"></script>  <!--模块化方式-->
<script>
    layui.use(['layer', 'form'], function () {  //模块化方式下，加载相关模块 
        var layer = layui.layer  //创建实例对象
            , form = layui.form;
        layer.msg('Hello Everyone'); //通过实例对象使用类方法，快显消息（稍后自动消失）
    });
    //layui.layer.msg('Hello Mr. Wu'); 
</script>
</body>

</html>
  • Layui框架对应于项目根目录下的文件夹layui
  • 需要分别引入Layui的css文件和js文件
  • 加载脚本文件layui.all.js对应于非模块化方式（此时所有模块都一次性加载），而加载layui.js对应于模块化方式
  • 本例js脚本只使用了layer模块，虽然也加载了form模块，但未使用

var data = ["北京",'上海','广州','深圳','杭州','长沙','合肥','宁夏','成都','西安','南昌','上饶','沈阳','济南','厦门','福州','九江','宜春',
            '赣州','宁波','绍兴','无锡','苏州','徐州','东莞','佛山','中山','成都','武汉','青岛','天津','重庆','南京','香港','澳门','台北'
];  //使用一个字符数组作为分页测试数据。其中，单双引号均可。
data.push('xxx');  //向数组添加数据
//实际项目里，数据来源于数据库服务器

layui.laypage.render({  //layui框架之分页模块laypage
    elem: 'nav'
    ,limit:5  //每页记录数，默认值为10
    , count: data.length
    , jump: function (obj) {  //重新导航时
        document.getElementById('city_list').innerHTML = function () {
            //数组连接方法concat()；删除方法splice()
            var arr = [], thisData = data.concat().splice((obj.curr-1) * obj.limit, obj.limit);
            //alert("当前页："+obj.curr);
            //alert(thisData); //当前页数据
            layui.each(thisData, function (index, item) {  //遍历
                arr.push('<li>' + item + '</li>');  //添加项目列表项
            });
            return arr.join('');   //将数组中的数据元素用自定义的连接符(空串，也可以是空格字符串)转换为字符串，必须！
        }();
    }
});
</script>
  • 在分页界面里，使用项目列表显示当面页数据
  • 对分页导航div定义jump事件来实现交互
  • 每页记录数的默认值为10，可重新设置
  • 本例中的数据存放在js数组里，所以是客户端分页方式；实际项目里，数据来源于数据库服务器，属于服务器端分页

4.2 http服务器的创建与访问

示例代码：

var http = require('http');  //加载http模块
var mysql = require("mysql");  

http.createServer(function(req, res){  //创建http服务器，req代表请求，res代表响应
    res.writeHead(200, {'Content-Type': 'text/plain; charset=utf-8'});  //固定用法
    //res.end("测试");
    var connection = mysql.createConnection({  //创建数据库连接对象   
        host     : 'localhost',       
        user     : 'root',              
        password : 'root',       
        port: '3306',                   
        database: 'memmana1', 
        dateStrings: true
    }); 
    connection.connect();  //打开数据库连接
    var  sql = 'SELECT * FROM user';
    connection.query(sql,function (err, result) {  //执行查询并返回结果
            if(err){
                res.end(err.message);
                return;
            }
            res.end(JSON.stringify(result));  //返回数据，从JS对象转化为JSon字符串
    });
    connection.end(); //关闭数据库连接
}).listen(3000);  //3000为监听端口
  • 在VS Code里进入命令行的另一种方法是：在项目文件夹空白处按住shift+右键，选择“在终端中打开”
  • 在项目的命令行方式下，执行node 2.js将创建一个Http服务器并运行
  • 在浏览器窗口请求 http://localhost:3000
  • 如果修改了源代码，则需要在终端窗口里按Ctrl+C停止http服务，然后重新创建

<!DOCTYPE html>

        <button v-on:click="getApi">调用ajax接口改变message</button>
        <p v-bind:title="title">鼠标悬停几秒钟查看此处动态绑定的提示信息！</p>
        <p v-if="seen">现在你看到我了</p>
        <ol>
            <li v-for="todo in todos" >
            {{todo.text}}
            </li>
        </ol>
    </div>
</body>
</html>

<script src="https://cdn.bootcss.com/jquery/3.2.1/jquery.min.js"></script>
<script src="https://unpkg.com/vue"></script>  <!--加载Vue.js-->
<script>
var app = new Vue({
    el: '#app',
    data: {
        message: 'Hello Vue!',
        title: '悬浮提示信息',
        seen: true,
        todos: [
            { text: '学习 JavaScript' },
            { text: '学习 Vue' },
            { text: '整个牛项目' }
        ]
    },
    methods: {
        reverseMessage: function () {
            this.message = this.message.split('').reverse().join('')
        },
        getApi:function() {
            var $this = this
            $.ajax({   //使用Ajax方法实现异步刷新
                url:'api.php',
                method:'get',
                success:function(data){
                    $this.message = data
                },
                error:function(err){
                    $this.message = '调用接口失败'
                }
            })
        }
    }
})
</script>

主页以Ajax方式请求的php页面api.php的代码如下：

<?php
    echo 'this is a string from api.php';
?>

  • Vue.js表现为一个js脚本库，类似于jQuery
  • Vue的核心库是关于视图层
  • MVVM的核心是视图与模型的双向绑定
